# 0x05강 - 스택

## 0x00 정의와 성질

- 스택 - 한쪽 끝에서만 원소를 넣거나 뺄 수 있는 자료구조 (FIFO: First In First Out)
- $O(1)$에 원소의 추가, 제거, 최상단의 원소 확인이 가능하다.

## 0x01 기능과 구현

- push, pop, top 기능

## 0x02 STL stack

- push, pop, top, empty, size 기능을 가지고 있음
- 스택이 비어있을 때 top이나 pop을 호출하면 런타임 에러가 발생
